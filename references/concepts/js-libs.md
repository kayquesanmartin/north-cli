# Catálogo de bibliotecas & ferramentas — JavaScript / TypeScript / React

Mapa lib/ferramenta → categoria → dificuldade (jr/pl/sr) usado pelos **insights
passivos** do north para ensinar as **bibliotecas, frameworks e ferramentas** que
aparecem no código (`import`, `package.json`, `npm install`). A IA casa o que usou contra
esta tabela; libs fora dela recebem dificuldade julgada pela IA (default `pl`). Namespace
do ledger: **`js-libs`**.
